# ぎんなんの小島と会話するツール

ローカルで動く、静的なチャットWebアプリです。AIやサーバ接続は不要。  
キャラクター「ぎんなんの小島」とブラウザ上で会話できます。

## 使い方
1. このフォルダをPCに保存
2. `index.html` をダブルクリックで開く（Chrome/Edge/Firefox など）
3. 画面上部で「モード / 臭気レベル / 敬語」を調整
4. 下部入力欄からメッセージ送信

## 機能
- モード切替：中立 / ラブコメ / ホラー / ルポ
- 臭気レベル：0〜5 （返答内の雰囲気が変化）
- 敬語⇄タメ口スライダ
- ルールベース応答（オフライン）
- 設定/lore は `lore.json` に記載。自由に改変OK

## 免責
- 本ツールは創作キャラクターとの模擬会話です。実在の人物・団体とは無関係です。
- ブラウザのみで動作し、個人データは外部送信しません。
